# Calendario

Genera calendarios PDF entre dos fechas desde la terminal.

## Instalación

(requiere python y pip)
usar el comando:

pip install calendario

## Uso

el comando calendario recibe dos argumentos:
el primero es la fecha de inicio del calendario
el segundo es la fecha de fin del calendario
la flag -m es opcional para ocupar solo media página

página completa (A4):
calendario DD-MM-AAAA DD-MM-AAAA

media página:
calendario DD-MM-AAAA DD-MM-AAAA -m

## Opciones

--help
--version
-m
